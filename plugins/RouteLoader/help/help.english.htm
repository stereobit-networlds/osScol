Fonction

Ce plugin est un exemple pour développer les vôtres. Son code est annoté afin d'en apprécier les règles de base.

Le résultat de ce plugin est très simple : il alloue aux matériaux liés dans l'ancre, une couleur de flat définie aléatoirement par le serveur.

Configuration

Il n'y a pas lieu de configurer ce plugin

Mise en oeuvre

Créez les liens sur les matériaux concernés et rassemblez-les dans une ancre.
Créez une instance de ce plugin en lui associant l'ancre.
Il n'y a aucun lien SCS à établir.
