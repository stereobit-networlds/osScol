Scale2 1.0

Category / Domain : C3D3 Plugin
SCOL Voy@ger Compatibility : V 3.2 or above
Update : 2003/02

Function

This module applies a change of scale in loop to the list of objects specified in the anchor. The result gives a throbbing effect.

The change of scale expresses itself in percentage of the size of every object and oscillate between a maximum size and a minimum size. The variation is linear (without acceleration) and depends of the display speed of the graphic card.

Configuration


Figure 1 : main interface of the Scale2 plugin


(1) ScaleFactorMax Maximum percentage (superior limit of the size of the object). Example: 200 for 200% of the initial size
(2) ScaleFactorMin Minimum percentage (lower limit of the size of the object). Example: 50 for 50% of the initial size
(3) ScaleFactorIncrement Throbbing speed. Example: 10 to vary from 10% the size of the object at each video cycle.
(4) Framerate Give the number of frames per second (more there are frames, more the movement will appear fluid);
if the computer is not powerful enough to display all the frames, some frames will be jumped
to guarantee that the movement won't be slower than foreseen (the value is by default 30).



Implementation

Create a link on the concerned object.
Create the anchor with these links.
Create the instance "Scale2 1.0" and associate him the anchor.
